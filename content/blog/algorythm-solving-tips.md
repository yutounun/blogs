## Set

```
class Solution(object):
    def containsDuplicate(self, nums):
        hset = set()
        for idx in nums:
            if idx in hset:
                return True
            else:
                hset.add(idx)
```

### When to use

Basically set is utilized when...

1. you look for a value in array real quick
2. you remove duplicated value

  Add element to set.
